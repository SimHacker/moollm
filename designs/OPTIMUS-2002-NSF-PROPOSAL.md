# OPTIMUS 2002: The NSF Proposal MOOLLM Answers

Upmanu Lall's 2002 NSF proposal (Columbia; Don Hopkins + David Levitt named consultants) specified
an educational city-sim platform SimCity couldn't yet be: open architecture, extractable state
variables, student-authored modules in a high-level language, multiple team roles, and — verbatim —
computers as "a **conversational environment** in which the learner can apply knowledge to problems
and consider their actions as reusable events."

MOOLLM is the conversational environment, literally: an LLM with the microworld as context.
Student modules become **skills** (natural language + YAML, legible to teachers and AI); "reusable
events" become git commits; the facilitator ships in the box as characters — the reproducibility
answer to constructionism's founder-in-the-room problem.
