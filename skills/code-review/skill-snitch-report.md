## Methods

| Method | Purpose |
|--------|---------|
| **START** | Begin a new code review |
| **EXAMINE** | Review specific file/section |
| **COMMENT** | Add review comment |
| **APPROVE** | Approve changes |
| **REQUEST-CHANGES** | Request fixes |
