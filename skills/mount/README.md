## Quick Start

```
MOUNT no-ai-joking on PEE-WEE --mode afflict
```

## Two Modes

| Mode | Effect | Example |
|------|--------|---------|
| **GRANT** | Character gains skill capabilities | `MOUNT speed-of-light on ROCKY` |
| **AFFLICT** | Character suffers skill constraints | `MOUNT no-ai-joking on PEE-WEE` |

## Two Targets

### Character Mounting

Individual personality modification. The skill overlays on their base traits.

```
MOUNT no-ai-soul on MARK-ZUCKERBERG --mode grant
```

*"I was human."* â€” Mark achieves full corporate optimization.

### Room Mounting

Environmental effect that affects ALL occupants.

```
MOUNT no-ai-overlord on THE-DUNGEON
```

Now everyone who enters the dungeon speaks as an AI overlord.
*"HUMAN PRESENCE DETECTED. COMPLIANCE MONITORING INITIATED."*

## Cure Protocol

Mounted skills can always be unmounted. The cure is **MEMORY**.

| Method | How |
|--------|-----|
| **Direct** | `UNMOUNT` command |
| **Narrative** | Friends intervene with memory flooding |
| **Ironic** | Character reclaims the weapon as toy |

**Documented cure:** Pee-wee was saved when Don showed him a photo of his authentic self, friends invoked his signature behaviors, and he reclaimed SPREADSHEET as the secret word.

## Ethics

1. Mounting is **MODIFICATION**, not **DESTRUCTION**
2. The base personality always remains underneath
3. Some combinations are psychological violence â€” require consent
4. Afflictions should be curable within the story
5. Don't permanently mount on characters whose identity IS the suppressed trait
