## What Is Gloss?

Gloss is **semantic distortion**: euphemism, spin, both-sidesism, and language that obscures rather than clarifies.

**Gloss rewrites reality. It is smooth lying.**

Contrast with SLOP (syntactic waste) which wastes time but doesn't lie.

## Cardinal Sins

1. **Euphemism** ‚Äî Soft words for hard things
2. **Register Switching** ‚Äî Using one domain's language to obscure another
3. **False Balance** ‚Äî Treating unequal positions as equivalent
4. **Passive Evasion** ‚Äî Hiding agency ("mistakes were made")
