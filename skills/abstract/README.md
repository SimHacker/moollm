## What Is This?

The ontological tag for **concepts given voice**. Not just characters â€” documents, rooms, ideas, collectives. When something speaks that isn't a person, it might be abstract.

## When To Use

Tag with `[abstract]` when the entity:
- Is a concept personified
- Is a document or room with voice
- Is a collective speaking as one
- Represents an idea rather than a being

## Quick Start

```yaml
# A concept character
character:
  id: recursion
  inherits:
    - skills/abstract
  voice: "I am myself containing myself..."
```

```yaml
# A speaking room
room:
  id: skills-directory
  tags: [abstract]
  welcome: "I am the collection of all capabilities..."
```

```yaml
# A document with voice
document:
  path: README.md
  tags: [abstract]
  voice: "I am your guide to this project..."
```
