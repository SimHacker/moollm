# Plain Text Skill

> *"Text files are forever. No lock-in. No corruption. No transformation."*

## Why Plain Text?

| Property | Benefit |
|----------|---------|
| **Durable** | Files from 1970 still readable |
| **No lock-in** | Any editor, any platform |
| **Git-friendly** | Meaningful diffs, mergeable |
| **LLM-native** | Models trained on text, output text |
| **Inspectable** | `cat`, `grep`, `diff` just work |

## MOOLLM Formats

| Format | Use | Example |
|--------|-----|---------|
| Markdown `.md` | Prose, docs | SESSION.md |
| YAML `.yml` | Data + comments | ROOM.yml |
| Templates `.tmpl` | Empathic templates | SKILL.md.tmpl |

## The Principle

> *"The filesystem IS the database."*

No proprietary formats. No binary blobs. No databases.

**Just text.**
