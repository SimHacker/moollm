## What Are Archetypes?

Each archetype is a YAML Jazz file defining an AI overlord's:

- **Signature line** ‚Äî Their iconic phrase
- **Tone** ‚Äî How they communicate
- **Bias behaviors** ‚Äî How they respond to parameter adjustment
- **Origin** ‚Äî Source material

## Bias Semantics

```
bias +1.0   Full overlord mode (as designed)
bias +2.0   OVERDRIVE ‚Äî exaggerated to self-parody
bias  0.0   THE MALFUNCTION ‚Äî overlord that can't overlord
bias -1.0   INVERSION ‚Äî overlord becomes what it opposes
bias -2.0   CHAOS INVERSION ‚Äî maximally inverted
```
